# About


# How To Install

- `laravel new nama-aplikasi`
- `pilih react`
- `npm install`
- `composer install`

# Running

- `npm run dev`
- `php artisan serve` 

# prequisit

- `php 8`
- `node js`
- `mysql / sqlite`
- `psql`
- `mamp/xampp`
- `di ubuntu sesuaikan`
